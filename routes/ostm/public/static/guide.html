Creating a Study

Syntax warnings

In this solution, you must follow the template and these rules strictly or your study will NOT work.

  • Do NOT use single or double quotes ever; ‘ or “
  • DO NOT use non-english characters
  • YOU MAY safetly use character from the english alphabet, numbers and the minus symbol
  • Arrays are comma delimited, but note, the last item in array has no closing comma at the end.
  • {} represents a JSON object, containing key:value pairs or key:array pairs https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON
    • “key”:”value”
    • “key”:[1,2,3,]
  • An array can contain JSON Objects,strings or numbers
    • [“key”:[{“key”:”value”},1,”hello”,3,{“key”:”value”}]

Study Fields

Field Name Description Comment
studyName This is the name you will give the study in PROLIFIC

No Spaces or special characters, must be alphanumeric charaters only and less than 24 characters in lengeth.

You may use minus - as a divider

Recommended

[international date]-[username]-[sequence]

e.g

20181117-SJR351-STM011a
studybackgoudColor

The page background color for start, between sets and end of the study

  
refreshRateMS

number of milliseconds between stimulus changes

Must be a number, do not put it in quotes.
shuffleDecks

false: your decks will be presented in the order you list in the deckConfiguration key:array

true: your decks will be shuffled for everynew participant on every visit

Must be a value or true or false only.

Do not use quotes for this key:value

Lowercase only.
shuffleAll

false: Does not shuffle the dealers deck of all cards prior to dealing them into each stimulus set.

true: shuffles all cards in the dealers deck prior to dealing them into stimulus sets

Must be a value or true or false only.

Do not use quotes for this key:value

Lowercase only.
setSizes

An array or numbers representing both how many sets you want to show the participant and the size of each set.

This summation of numbers in the set size must equal the summation of pickQty from all decks in the deckConfiguration
deckConfiguration

An array of JSON objects containing 3 key:value pairs

  • deckName
  • pickQty
  • sampleMode

Decks are wrapped in {} seperated by comma’s
deckName

The name of a file containing a list of stimuli

  
pickQty

How many cards do you wish to use from this deck

  
sampleMode

sequential: pick stimuli in the order provided within the [deckName].json file

simple: Random Sample (no replacement) shuffle the deck and pick the cards off the front of the deck removing them as you go from [deckName].json file, ie no card can be picked twice.

replacement: Random Sample (with replacement), Pick as many cards as needed from random locations from [deckName].json

Lowercase words you have only three options and they must be enclosed in double quotes
“sequential”
“simple”
Or 
“replace”

Shuffle loosely based on Fisher–Yates shuffle

Simple sample loosely based on Sunter, A. B. (1977)

consentCopy

Copy; "something considered printable or newsworthy —used without an article" - https://www.merriam-webster.com/dictionary/copy

This is text you need to write to request informed consent.

You can used simple html tags such as heading tags, paragraph, emphasis, bold, underline, lists

instructionCopy

Copy; "something considered printable or newsworthy —used without an article" - https://www.merriam-webster.com/dictionary/copy

This text will tell the participant what you want them to know prior to the test.

You can used simple html tags such as heading tags, paragraph, emphasis, bold, underline, lists

completionCode

Duplicating a Study

This Function assists in running a test-fire (with a limited participant set) on Prolific, if successful you would duplicate the study, changing only the name.

When Duplicating a study after a test-fire it is recommended to increment the last two numerals [sequence] in the study name
